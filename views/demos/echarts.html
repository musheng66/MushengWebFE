ECharts

ECharts，一个纯 Javascript 的图表库，可以流畅的运行在 PC 和移动设备上，兼容当前绝大部分浏览器（IE8/9/10/11，Chrome，Firefox，Safari等），底层依赖轻量级的 Canvas 类库 ZRender，提供直观，生动，可交互，可高度个性化定制的数据可视化图表。

Github地址： https://github.com/ecomfe/echarts
官网地址： http://echarts.baidu.com/

核心代码

<!DOCTYPE html>
<html>
<head>
</head>
<body>
<div id="main" class="" style="width:100%;height:300px;"></div>
<!-- ECharts plugin -->
<script type="text/javascript" src="echarts.min.js"></script>
<script type="text/javascript">
//配置
var option = {
    title : {
        text: '',
        subtext: ''
    },
    tooltip : {
        trigger: 'axis'
    },
    toolbox: {
        show : true,
        feature : {
            dataView : {show: true, readOnly: false},
            magicType : {show: true, type: ['line', 'bar']},
            restore : {show: true},
            saveAsImage : {show: true}
        }
    },
    xAxis : [
        {
            type : 'category',
            boundaryGap : false,
            data : <{if $xAxisdata}><{$xAxisdata}><{else}>['无数据']<{/if}>
        }
    ],
    yAxis : [
        {
            type : 'value',
            axisLabel : {
                formatter: function(value){
					return value + ' 次';
				},
			},
        }
    ],
    series : [
        {
            name:'启动次数',
            type:'line',
            data:<{if $seriesdata}><{$seriesdata}><{else}>[0]<{/if}>,
            markPoint : {
                data : [
                    {type : 'max', name: '最大启动次数'},
                    {type : 'min', name: '最小启动次数'}
                ]
            },
           itemStyle:
            {
                normal: {
                     lineStyle: {
                       color : 'skyblue'
                     }
                }
            },
        }
    ],

};
$(function(){
    //配置：
    var myChart = echarts.init(document.getElementById('main'));
	myChart.setOption(option);
});
</script>
</body>
</html>

页面源码

HTML

1.echarts并不基于jquery，可独立使用，仅需引入ECharts插件。

html代码：

<!-- ECharts plugin -->
<script type="text/javascript" src="echarts.min.js"></script>

2.在页面中加入一个div，以便配置显示ECharts。

html代码：

<div id="mainChart1" class="ECharts" style="height: 500px; width: 500px;"></div>

Javascript

1.可在页面<javascript>标签中配置ECharts。

JS代码：

option1 = {
    title: {
        text: '世界人口总量',
        subtext: '数据来自网络'
    },
    tooltip: {
        trigger: 'axis',
        axisPointer: {
            type: 'shadow'
        }
    },
    legend: {
        data: ['2011年', '2012年']
    },
    grid: {
        left: '3%',
        right: '4%',
        bottom: '3%',
        containLabel: true
    },
    xAxis: {
        type: 'value',
        boundaryGap: [0, 0.01]
    },
    yAxis: {
        type: 'category',
        data: ['巴西','印尼','美国','印度','中国','世界人口(万)']
    },
    series: [
        {
            name: '2011年',
            type: 'bar',
            data: [18203, 23489, 29034, 104970, 131744, 630230]
        },
        {
            name: '2012年',
            type: 'bar',
            data: [19325, 23438, 31000, 121594, 134141, 681807]
        }
    ]
};

$(function() {
    //配置
    var myChart1 = echarts.init(document.getElementById('mainChart1'));
    myChart1.setOption(option1);
});

注意事项

1.ECharts 在初始化图表的时候，选定的展示图表的div标签必须设定固定的width和height属性值，否则其内层标签无长宽，图表无法显示。
